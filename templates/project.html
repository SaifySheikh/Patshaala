Project Description

       
       

Portfolio

The Weather Detection Web Application is a user-friendly and informative web-based tool that allows users to quickly access real-time weather information for any location worldwide. This project utilizes a combination of HTML, CSS, and JavaScript to create an interactive and visually appealing interface. Additionally, it leverages a weather API to fetch and display up-to-date weather data.

Tech Stack -

  • HTML
  • CSS
  • JAVASCRIPT
  • WEATHER API
